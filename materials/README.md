# AI 실습 연수 수업 원고 자료

중학교 선생님 대상 **50분 생성형 AI 실습 연수**용 슬라이드입니다.  
웹 가이드(`htmls/index.html`)의 **내용·프롬프트·실습 절차**를 슬라이드에 담았습니다.

## 파일

| 파일 | 설명 |
|------|------|
| `AI_lecture_script.pptx` | PowerPoint 원본 (**18매**, 발표자 노트 포함) |
| `AI_lecture_script.pdf` | PDF 출력물 (배포·인쇄용) |
| `images/` | 슬라이드용 인포그래픽 이미지 |
| `generate_lecture_slides.py` | PPT·PDF·이미지 재생성 스크립트 |

## 디자인

- **밝은 배경** (흰색·연한 파랑) — 가독성 우선
- **큰 글씨** (본문 22~24pt, 제목 34~48pt)
- **인포그래픽 이미지** (로드맵, 흐름도, 도구 선택법 등)

## 슬라이드 구성 (18매)

1. 표지
2. 오늘의 목표 & 50분 로드맵 (이미지)
3. 3분 빠른 시작 (이미지)
4. 광주아이온 — 통합 인증 & 접속 (이미지)
5. 광주아이온 — 학습 분석 대시보드
6. NotebookLM 소개 (이미지)
7. 1-1 학교 맞춤 챗봇 + **프롬프트**
8. 1-2 세특 초안 + **프롬프트**
9. 1-3·1-4 평가계획·문항 + **프롬프트**
10. 교무기획부·교육연구부 + **프롬프트**
11. 담임 — 학부모·학급활동 + **프롬프트**
12. Gemini @기능 & Gems + **프롬프트**
13. Google Forms 자동화 + **프롬프트**
14. 슬라이드 7단계 (이미지 + **프롬프트**)
15. 워크스페이스+ + **프롬프트**
16. 2022 개정 교육과정 + **프롬프트**
17. 상황별 활용 가이드 (이미지)
18. 마무리 & 주의사항 3가지

각 슬라이드 **발표자 노트**에 수업 원고(멘트)가 포함되어 있습니다.

## 재생성

```bash
pip install python-pptx reportlab pillow

python3 generate_lecture_slides.py
```
